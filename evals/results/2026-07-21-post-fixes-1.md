(22V406000, 24V838000, 18E097000, 23V411000, 22V426000) — was **wrong**. Live reproduction

`grading_notes` listed only 4 of the 11 real campaigns, so it flagged the 7 unlisted-but-real ones
as "invented." The deterministic check passed rec_05 correctly; we then trusted the wrong layer.
**Fixed** by moving grounding to the deterministic layer (which holds the tool results as ground
truth): a `\d{2}[VE]\d{6}` token in the answer that is absent from this turn's tool results is now
a hard veto — a genuine hallucination is *structurally* caught, while real-but-unlisted campaigns
are not. See the CORRECTION note in the rec_05 failure block above. **POSTMORTEM candidate** — the
lesson is layered grading: an LLM judge asked to verify grounding without the source data will
invent violations, and a green deterministic check must not be overridden by an unfounded judge fail.
